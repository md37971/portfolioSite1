Small game developed in Pygame.

DiceRollUI

UI Project developed in Java & Python.

A UI program developed in Apache NetBeans to take the sum of three dices per iteration to measure the frequency of numbers. Additionally, a simple add-on program developed in Python is included to display bar graphs of the data from the Java UI project.
