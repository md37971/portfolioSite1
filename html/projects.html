My Projects!

Seeing what people would make in computers was always something that fascinated me, especially playing fun video games, which has inspired me multiple times as a kid to develop projects on my own, though at times this was difficult learning by myself. Fortunately, college has helped me learn about the fundamentals of programming, which helped me develop these fun side projects in my free time!

All of the projects listed here can be viewed on my GitHub!

Alien Delivery

Developed in Python using the Pygame Library.

You're an alien who works in a delivery job, and you must deliver the package on time before it's late; however, there are dangerous obstacles in the sky. You still must deliver the package regardless.

DiceRollUI

UI Project developed in Java & Python.

A UI program developed in Apache NetBeans to take the sum of three dices per iteration to measure the frequency of numbers. Additionally, a simple add-on program developed in Python is included to display bar graphs of the data from the Java UI project.
