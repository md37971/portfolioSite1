My Projects!

Seeing what people would make in computers is always something that fascinated me, especially playing fun video games, which has inspired me multiple times as a kid to develop projects on my own, though at times this was difficult learning by myself. Fortunately, college has helped me learn about the fundamentals of programming, which helped me develop these fun side projects in my free time!

Most of these projects listed here can be viewed on my GitHub!

Alien Delivery

Developed in Python using the Pygame Library.

Synopsis: You're an alien who works in a delivery job, and you must deliver the package on time before it's late; however, there are dangerous obstacles in the sky. You still must deliver the package regardless.

  • Uses the Pygame library to implement game mechanics in Python.
  • Art is originally created using Paint.net.

DiceRollUI

UI Project developed in Java & Python.

  • A UI program developed using Apache Netbeans IDE in Java.
  • Works by taking the sum of three dices per iteration by the number of times the user wishes for measuring the frequency of numbers.
  • Ability to save results in a '.csv' format.
  • Bundled with a Python program to display a bar graph of resulting data using the MatPlotLib library.

UAFS Rotaract Bot

Developed in Python using Discord & Google Sheets API.

  • A simple Discord bot created for the purpose to add interaction to the UAFS Rotaract Discord server.
  • Implemented commands such as a leaderboard viewer, event announcer, and social media lister.
  • Included automatic features such as an autowelcomer for new members and an autoban feature to prevent spam accounts.
  • Made possible by using Discord's API to read server data and Google Sheets API for viewing the leaderboard, which is all written in Python.
  • Ran 24/7 using Render's hosting service.
